### 2. FIX REACT HOOK DEPENDENCIES

For ALL useEffect hooks:

* Add ALL required dependencies to dependency array
* If adding dependencies causes re-renders, fix properly using:

  * useCallback
  * useMemo

DO NOT ignore warnings.

BAD:
useEffect(() => {
fetchData();
}, []);

GOOD:
useEffect(() => {
fetchData();
}, [fetchData]);

OR memoize fetchData if needed.
